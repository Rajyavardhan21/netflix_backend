To Run this Project - Follow below steps.
   ----------------------------------


1. Open cmd prompt or any other terminal

2. Use command - 'mongosh' - To run mongo shell.

3. Use command - 'use movies_db' - To Create a Database.

4. Use command - 'db.movies.insertMany([ <- data (objects) from movies.json file -> ])' 
				 - To create a collection and add data to it.
	           
5. Open current folder with Visual Studio Code App

6. Open the VS code terminal

7. Use command - 'npm init' - to install the npm

8. Install all the dependencies

9. Use command - 'nodemon app.js' - To run the server on nodemon app

10. Server is Running succesfully now!.

11. Check these different urls --> 

		For example these urls help to,
		1. Fetch the list of all the movies - http://localhost:8008/shows/      
		2. Fetch the details of given id    - http://localhost:8008/shows/63ca94033966fc9d52119651/  
		3. Streams the movie by chunks      - http://localhost:8008/shows/63ca94033966fc9d52119651/video
